# GitHub Bulk Upload 🚀
A lightweight tool for uploading multiple files or folders to GitHub in one go.

Key Features:
* Built using Kotlin Multiplatform (KMP)
* Compose Multiplatform (UI structure implied by `composeApp/`)

What This Project Does:
This tool provides a simple way to automate bulk uploads. Instead of manually adding files through the GitHub UI, it:
* Uploads multiple files or folders at once
* Automates commit + push
* Works with nested directories
* Reduces repetitive manual steps

## How It Works (High-Level)
1. Select or prepare the files/folders
2. The tool processes them
3. A commit is generated
4. Changes are pushed to the chosen repository
